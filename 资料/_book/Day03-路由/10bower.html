Introduction

Bower

  • Bower是1个前端库或者框架市场,通过Bower可以下载前端相关的库或者框架.和npm一样.
  • Bower依赖于NodeJs和Git
  • 并且通过npm下载安装Bower
  • 全局安装Bower npm install bower -g
  • 使用Bower下载库或者框架.
    • 在指定目录打开git bash.
    • 使用命令bower install 库或者框架名称 就可下载指定插件. 你观察log信息应该清楚其实际上是去gitgub下载.
    • 会在当前目录中 生成1个bower_components目录,将下载的库或者框架放在其中.
  • 下载指定版本的库
    • bower install jquert#1.9.1
  • 查看库的信息
    • bower info jquery
  • 查询库信息
    • bower search validate 查询和validate相关的库
  • 删除库
    • bower uninstall 库名
  • 具有依赖关系的库,bower会自动下载
  • 如果删除被依赖的库,将无法删除.
